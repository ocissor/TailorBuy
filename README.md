Build TailorBuy_Temp from the source and intsall dependencies:

1. **Clone the repository:**

    ```sh
    ❯ git clone ../TailorBuy_Temp
    ```

2. **Navigate to the project directory:**

    ```sh
    ❯ cd TailorBuy_Temp
    ```

3. **Install the dependencies:**
